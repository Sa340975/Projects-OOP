# Project 1 – Object-Oriented Programming (Java)

## Student Marks Management System

This project is part of the **Object-Oriented Programming (OOP)** course.  
It demonstrates the basic use of Java and OOP concepts to manage student marks.

---

## 🎯 Project Objectives
- Practice **basic Java programming**
- Apply **OOP principles**
- Handle student data and marks
- Perform calculations on marks

---

## 🧩 Features
- Enter student information (Name, ID)
- Enter marks for multiple courses
- Calculate:
  - Average
  - Highest mark
  - Lowest mark
- Determine **Pass / Fail**
- Display student details and marks information

---

## 🛠 Concepts Used
- Classes & Objects
- Encapsulation
- Arrays / ArrayList
- Conditional statements
- Loops
- Basic exception handling
